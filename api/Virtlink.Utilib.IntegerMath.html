    Modulo(Int32, Int32)

    Calculates the dividend modulo the divisor.

    Declaration
    public static int Modulo(int dividend, int divisor)
    Parameters
    Type Name Description
    System.Int32 dividend

    The dividend.
    System.Int32 divisor

    The divisor.
    Returns
    Type Description
    System.Int32

    The result of dividend modulo divisor.
    Remarks

    When dividend is positive, this method performs the same operation as dividend % divisor. However, when dividend is negative, this method returns the modulo instead of the remainder. The result will therefore always be positive.

    The sign of the divisor is ignored.

    Exceptions
    Type Condition
    System.DivideByZeroException

    The divisor cannot be zero.
