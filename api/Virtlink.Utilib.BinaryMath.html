    RoundToNextPowerOfTwo(Int32)

    Rounds the specified value to the next power of two.

    Declaration
    public static int RoundToNextPowerOfTwo(int value)
    Parameters
    Type Name Description
    System.Int32 value

    The value to round.
    Returns
    Type Description
    System.Int32

    The next power of two.
    Remarks

    For positive values, this function returns the next higher power of two. For the value 0, this function returns 1. For negative values, this function returns the next lower power of two.

    Exceptions
    Type Condition
    System.OverflowException

    value is greater than 0x40000000.
